Track every job application, auto-extract job details from 
any posting URL, and see exactly how well your skills match 
each role — all in one place.

## Features
- JWT authentication (register/login/logout)
- Paste any job URL → auto-extracts title, company, 
  skills, salary, location via web scraping
- Track applications through 8 stages from Saved → Offer
- Skill gap analysis — see your % match and missing skills
- Dashboard with funnel chart and pipeline stats
- Automated follow-up email reminders
- Full REST API with Swagger documentation

## Tech Stack
